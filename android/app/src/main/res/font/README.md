# Inter Font Family

Download Inter font files from https://fonts.google.com/specimen/Inter

Required font files:
- `inter_regular.ttf` (Regular / 400 weight)
- `inter_medium.ttf` (Medium / 500 weight)
- `inter_semibold.ttf` (SemiBold / 600 weight)
- `inter_bold.ttf` (Bold / 700 weight)

Place the downloaded .ttf files in this directory.

Fallback: If Inter is not available, the app will use system Roboto font.
